# 📊 Overview

Finance Dashboard is a modern web application built using React and Vite that provides interactive visualization of financial data such as income, expenses, and savings.

It transforms raw financial data into meaningful insights through dynamic charts and a responsive interface, enabling better financial decision-making.

---

# 🚀 Key Features

# 📈 Interactive Charts
Visualize financial data using Recharts for better understanding.

# 💵 Income vs Expense Tracking
Track and compare income and expenses effectively.

# 📊 Category-wise Analysis
Analyze spending patterns across different categories.

# 📉 Savings Insights
Monitor savings trends and financial health.

# ⚡ Fast Performance
Optimized performance using Vite build system.

# 🎨 Modern UI
Clean and responsive design using Tailwind CSS.

# 🌐 Deployment
Deployed on Vercel for global accessibility.

# ⚙️ Installation & Setup

# Step 1: Clone Repository
git clone https://github.com/Prathamesh196/Finance-Dashboard.git

# Step 2: Navigate to Project Folder
cd Finance-Dashboard

# Step 3: Install Dependencies
npm install

# Step 4: Run Development Server
npm run dev

---

# 🏗️ Build for Production

npm run build
